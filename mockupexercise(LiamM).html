(H3)This exercise will challenge you to apply some basic color styling to a webpage through a combination of style sheets:

  • External style sheets
  • Embedded styles
  • Inline styles

    • (p) Your task is as follows:

    1. Recreate the basic HTML structore of this webpage from scartch
    2. Create an external cascading syle sheet and link it in the document
      • Create style rules in your CSS file to style the HTML elements as appropriate
        • Visit Coolors and experiment with generating a color scheme for the site
        • <p> tags should have an RGB color value applied to them
        • <H> tags should be styled using an embedded style

        • <a> tags should be styled using inline styles
