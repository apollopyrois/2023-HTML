Sectioning Elements

Basics of Sectioning Elements

Sectioning elements help us to map out the core pieces of our website. We can think of sectioning elements as containers that will contain site content

Examples of Sectioning Element tags

<address>, <article>, <aside>, <body>, <footer>, <h1-h6>, <header>, <nav>, <section>

Grouping Elements

Basics of Grouping Elements

Grouping elements are contained within sectioning elements

For example, a list of items showing different grouping element tags

  • <ol> for an ordered list of items
  • <ul> for an unordered list of items
  • <p> for text of a paragraph
  • <blockquote> for a quotation, often used with a citation

Text-Level Elements

Basics of Text-Level Elements

Text-level elements are distinct from sectioning and grouping elements because they are known as "inline" elements. They are embedded within the content of other elements
